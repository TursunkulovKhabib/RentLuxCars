#  RentLuxCars — Premium Car Rental Platform

**RentLuxCars** — это современное веб-приложение (SPA) для аренды автомобилей премиум-класса. Проект разработан с использованием передовых технологий фронтенда 2025 года и облачной базы данных Supabase. Платформа позволяет пользователям просматривать каталог элитных автомобилей, а зарегистрированным владельцам — управлять своими объявлениями.

##  Как пользоваться сайтом (Сценарии)

### 1. Гость (неавторизованный пользователь)
- **Изучение главной страницы**: На главном экране вас встречает эффектный слайдер с флагманскими моделями и три карточки с категориями: "Executive", "Sports" и "Family".
- **Переход в каталог**: Клик по любой из карточек категорий моментально перенесет вас в общий каталог и применит соответствующий фильтр.
- **Просмотр каталога**: На странице каталога можно фильтровать машины по категориям или просматривать все доступные предложения.
- **Контакты**: Через боковое меню можно перейти на страницу "Контакты", чтобы увидеть информацию о менеджерах.

### 2. Регистрация нового пользователя
- **Переход на страницу регистрации**: Нажмите на "Бургер-меню" (☰) и выберите "Войти в аккаунт", затем на странице входа кликните по ссылке "Зарегистрироваться".
- **Заполнение формы**: Введите ваше имя, email и пароль (минимум 6 символов).
- **Подтверждение по почте**: После регистрации Supabase отправит вам на почту письмо со ссылкой для подтверждения аккаунта. **Вход будет невозможен до перехода по этой ссылке.**

### 3. Авторизованный пользователь (владелец авто)
- **Вход в аккаунт**: После подтверждения почты, войдите на сайт через страницу `/login`.
- **Личный кабинет "Мои объявления"**: После входа вас автоматически перенаправит в личный кабинет (`/admin`), где отображаются только ваши объявления. В боковом меню также появится пункт "Мои объявления".
- **Управление объявлениями**: В личном кабинете вы можете видеть свои активные объявления и добавлять новые. Есть вкладка для просмотра архивных объявлений.
- **Создание нового объявления**: Нажмите на кнопку "Создать объявление". Откроется удобная форма с живым превью, где вы можете указать марку, модель, цену, категорию и загрущить фото. После публикации объявление появится в вашем личном кабинете и в общем каталоге.
- **Выход из аккаунта**: В личном кабинете есть кнопка "Выйти".

##  Технологический стек

### Frontend
*   **React 18**: Основная библиотека для построения UI.
*   **TypeScript**: Строгая типизация для надежности и масштабируемости.
*   **Redux Toolkit**: Глобальное управление состоянием (авторизация, каталог, фильтры).
*   **React Router v7**: Клиентская маршрутизация с умной загрузкой сессии.
*   **Tailwind CSS**: Утилитарный CSS-фреймворк для быстрой и адаптивной верстки.
*   **Lucide React**: Легкие и кастомизируемые иконки.

### Backend & Infrastructure
*   **Supabase**:
    *   **PostgreSQL**: Реляционная база данных для хранения авто и пользователей.
    *   **Auth**: Готовая система аутентификации (Email/Password, JWT, RLS).
    *   **API**: Автоматически генерируемый RESTful API.

## Инструкция по установке и запуску

Для запуска проекта вам понадобится установленный **Node.js** (версии 16 или выше) и **npm**.

### 1. Клонирование репозитория
Откройте терминал (PowerShell, CMD или Terminal на macOS/Linux) и выполните команду:
  `git clone https://github.com/TursunkulovKhabib/RentLuxCars.git`
### 2. Установка зависимостей
Установите все необходимые библиотеки одной командой:
  `npm install`
### 3. Запуск проекта
Запустите локальный сервер разработки:
  `npm start`
После этого откройте браузер и перейдите по адресу [http://localhost:3000](http://localhost:3000).

## Структура проекта

*   **`src/api`** — Конфигурация клиента Supabase и методы API.
*   **`src/components`** — Переиспользуемые UI компоненты (Button, Input, Slider).
*   **`src/features`** — Redux Slices (логика данных и авторизации).
*   **`src/pages`** — Основные страницы (Home, Catalog, CarDetails, Admin, CreateAd).
*   **`src/store`** — Настройка Redux Store.
*   **`src/types`** — Глобальные TypeScript интерфейсы.
